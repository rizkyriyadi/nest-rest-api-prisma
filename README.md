## Description

Rest API in nestjs with swagger

## Installation

First install all the dependency

```bash
$ npm install
```

## Running the app

```bash
# run the docker frist to init the database
$ docker-compose up

# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
